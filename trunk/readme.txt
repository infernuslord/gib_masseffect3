If you can't edit any collections in the Raw tab in SaveEdit, you probably
don't have the full .NET Framework 4 installed, but have the Client Profile
installed.

To fix this, install the full .NET Framework 4.

http://www.microsoft.com/download/en/details.aspx?id=17851
